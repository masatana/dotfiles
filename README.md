# My dotfiles

## How to use

    $ mkdir ~/dotfiles
    $ git clone git@github.com:masatana/dotfiles.git ~/dotfiles
    $ cd dotfiles
    $ chmod 700 setup.sh
    $ ./setup.sh
    $ vim ~/.vimrc

## Note
If you wanted to write sensitive information, please write it in .vimrc.local
