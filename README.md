# My dotfiles (no dependency version)

## How to Setup

```sh
$ mkdir ~/dotfiles
$ git clone git@github.com:masatana/dotfiles.git ~/dotfiles
$ cd dotfiles
$ chmod 700 setup.sh
$ ./setup.sh
$ vim ~/.vimrc
```

And read [this](https://github.com/junegunn/vim-plug)

or, just copy and paste ;)

## How to Uninstall

```sh
$ ./setup.sh -u
```
